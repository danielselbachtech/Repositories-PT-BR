# Windows em Ambientes Virtualizados

Este subrepositório é dedicado a conteúdos relacionados à execução do Windows Server em ambientes virtualizados.

## Plataformas abordadas

- VMware
- Hyper-V
- Outros hypervisors compatíveis

## Conteúdo esperado

- Boas práticas de configuração
- Integração com ferramentas do hypervisor
- Considerações de performance
- Pontos de atenção em auditorias
- Problemas comuns e soluções

## Público-alvo

- Infraestrutura
- Administradores de virtualização
- Suporte avançado
