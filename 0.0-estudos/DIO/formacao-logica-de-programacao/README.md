# Formação Lógica de Programação

Formação oferecida pela [DIO](https://www.dio.me/) com foco nos fundamentos da programação. Ao longo da formação aprendi a trabalhar com variáveis, laços de repetição, funções e objetos, construindo uma base sólida antes de aprender qualquer linguagem. Os exemplos e desafios foram realizados com JavaScript.

## Conteúdo da Formação

| Módulo | Descrição |
|---|---|
| Bem-Vindo à Sua Jornada | Introdução e materiais complementares |
| Introdução à Lógica de Programação | Algoritmos, fluxogramas, variáveis, vetores e matrizes |
| Explorando Operadores | Operadores aritméticos, de comparação e lógicos |
| Dominando Estruturas de Controle | if/else, switch case, estruturas de repetição |
| Trabalhando com Funções | Funções, parâmetros e retorno |
| Refinando Técnicas | Desafios de código práticos |
| Estruturas de Dados e Objetos | JSON, classes e objetos |

## Desafios de Projeto

### Desafio 01 — Classificador de Nível de Herói

Desafio de lógica de programação da DIO onde criei um sistema que classifica um herói em diferentes níveis, do Ferro ao Radiante, com base na quantidade de XP que ele possui. O sistema também trata casos inesperados, como um XP inválido, retornando o nível "Desconhecido". Foi meu primeiro contato com estruturas de decisão em JavaScript e aprendi como o computador pode tomar decisões automaticamente dependendo de um valor.

[Ver código](./desafios-de-projetos/desafio-01/src/desafio-01.js)


---

### Desafio 02 — Calculadora de Partidas Rankeadas

Desafio de lógica de programação da DIO onde criei um sistema que calcula o saldo de partidas rankeadas (vitórias menos derrotas) e classifica o nível do herói do Ferro ao Imortal. Foi meu primeiro contato prático com funções e laços de repetição em JavaScript, onde aprendi a isolar a lógica do cálculo e automatizar testes de vários resultados de uma única vez.

[Ver código](./desafios-de-projetos/desafio-02/src/desafio-02.js)


---

### Desafio 03 — Escrevendo as classes de um Jogo

Desafio de lógica de programação da DIO onde criei um sistema de classes que representa heróis de uma aventura com propriedades de nome, idade e tipo. Foi meu primeiro contato prático com Programação Orientada a Objetos em JavaScript, onde aprendi a utilizar o método de ataque dinâmico para cada classe e a manipular as propriedades internas do objeto através do ponteiro `this`.


[Ver código](./desafios-de-projetos/desafio-03/src/desafio-03.js)
