## Desafio 01 — Classificador de Patentes

Desafio de lógica de programação da DIO onde criei um sistema que classifica um herói em diferentes níveis, do Ferro ao Radiante, com base na quantidade de XP que ele possui. O sistema também trata casos inesperados, como um XP inválido, retornando o nível "Desconhecido". Foi meu primeiro contato com estruturas de decisão em JavaScript e aprendi como o computador pode tomar decisões automaticamente dependendo de um valor.

[Ver código](./desafio-01/src/desafio-01.js)

### Desafio 02 — Calculadora de Partidas Rankeadas

Desafio de lógica de programação da DIO onde criei um sistema que calcula o saldo de partidas rankeadas (vitórias menos derrotas) e classifica o nível do herói do Ferro ao Imortal. Foi meu primeiro contato prático com funções e laços de repetição em JavaScript, onde aprendi a isolar a lógica do cálculo e automatizar testes de vários resultados de uma única vez.
